## Installation

`git clone git://github.com/joshlapham/dotfiles.git ~/.dotfiles && cd ~/.dotfiles && rake`

OR for the bash script way -

`git clone git://github.com/joshlapham/dotfiles.git ~/.dotfiles && cd ~/.dotfiles && bash bootstrap.sh`
